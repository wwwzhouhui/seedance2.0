## 2. 产品概述

### 2.1 产品定位

Seedance 2.0 是一款基于字节跳动即梦平台 Seedance 2.0 模型的 AI 视频生成 Web 应用。产品面向内容创作者、设计师、营销人员等用户群体，提供简单易用的图生视频功能。

### 2.2 核心价值

- **降低视频制作门槛**：无需专业视频制作技能，通过自然语言描述即可生成高质量视频
- **多图参考能力**：支持上传多张参考图片，实现更精准的视频内容控制
- **灵活配置**：提供多种画面比例和时长选择，适应不同场景需求

### 2.3 目标用户

| 用户群体 | 使用场景 |
|----------|----------|
| 内容创作者 | 社交媒体短视频、创意内容制作 |
| 设计师 | 动态海报、视觉概念展示 |
| 营销人员 | 产品宣传视频、广告创意 |
| 个人用户 | 生日祝福、纪念视频制作 |

### 3.1 功能列表

| 功能模块 | 功能项 | 优先级 | 说明 |

| 图片管理 | 图片上传 | P0 | 支持点击和拖拽上传，最多5张 |

| 提示词 | 文本输入 | P0 | 支持5000字符，支持@引用语法 |
| 视频配置 | 参考模式选择 | P0 | 全能参考、首帧参考、尾帧参考 |
| 视频配置 | 画面比例选择 | P0 | 6种预设比例 |
| 视频配置 | 时长选择 | P0 | 4-15秒可选 |

| 系统设置 | SessionID 配置 | P0 | 支持环境变量和界面配置 |

### 3.2 功能详细说明

#### 3.2.1 图片上传

**输入约束：**
- 支持格式：常见图片格式 (JPEG, PNG, GIF, WebP 等)
- 数量限制：1-5张
- 单文件大小：最大 20MB

**交互流程：**
1. 用户点击上传区域或拖拽图片到指定区域
2. 系统验证图片格式和大小
3. 显示缩略图预览，自动分配引用索引 (@1, @2, ...)
4. 用户可通过点击删除按钮移除单张图片

#### 3.2.2 提示词输入

**语法规范：**
- 普通文本描述视频场景
- 使用 `@1`, `@2` 等占位符引用对应图片
- 示例：`@1作为首帧，@2作为尾帧，人物向前行走`

**字符限制：** 最大 5000 字符

#### 3.2.3 参考模式

| 模式 | 说明 |
|------|------|
| 全能参考 | AI 智能识别参考图片内容，综合生成视频 |
| 首帧参考 | 以参考图片作为视频首帧进行延伸 |
| 尾帧参考 | 以参考图片作为视频尾帧进行逆向生成 |

#### 3.2.4 画面比例

| 比例 | 适用场景 | 分辨率 |
|------|----------|--------|
| 21:9 | 电影宽屏 | 1680×720 |
| 16:9 | 标准横屏视频 | 1280×720 |
| 4:3 | 传统屏幕 | 960×720 |
| 1:1 | 社交媒体正方形 | 720×720 |
| 3:4 | 竖屏内容 | 720×960 |
| 9:16 | 手机竖屏/短视频 | 720×1280 |

#### 3.2.5 视频时长

- 可选范围：4-15 秒
- 默认值：5 秒

#### 3.2.6 视频生成流程

```
用户提交 → 后端上传图片 → 调用即梦API → 后端轮询状态 → 返回视频URL
```

**状态反馈：**
- 提交中：显示提交进度
- 生成中：显示等待时间和当前状态
- 成功：自动播放视频
- 失败：显示错误信息，支持重试

## 4. 非功能需求

### 4.1 性能要求

| 指标 | 要求 |
|------|------|
| 页面加载时间 | < 3秒 |
| 图片上传响应 | < 2秒 |
| 视频生成时间 | 取决于即梦API，通常2-5分钟 |
| 并发支持 | 单用户串行生成 |

### 4.2 兼容性要求

| 类型 | 支持范围 |
|------|----------|
| 浏览器 | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ |
| 操作系统 | Windows 10+, macOS 11+, 移动端浏览器 |
| 屏幕尺寸 | 响应式设计，支持移动端 |

### 4.3 安全要求

- SessionID 不在 URL 或日志中明文显示
- 支持 SessionID 输入框密码隐藏
- 敏感配置通过环境变量管理

### 4.4 可用性要求

- 界面简洁直观，符合 Material Design 设计原则
- 关键操作有明确反馈
- 错误信息友好易懂

## 5. 用户体验设计

### 5.1 界面布局

采用左右分栏布局：
- **左侧面板**：配置区域（图片上传、提示词、参数选择、生成按钮）
- **右侧面板**：视频预览区域

### 5.2 视觉风格

- 深色主题 (#0f111a 背景)
- 紫色主色调 (purple-600)
- 圆角卡片设计
- 平滑过渡动画

### 5.3 交互细节

| 交互 | 反馈 |
|------|------|
| 按钮点击 | 渐变高亮效果 |
| 图片上传成功 | 缩略图淡入动画 |
| 生成进度 | 进度条动画 + 文字状态 |
| 错误提示 | 红色边框 + 错误文案 |
